## Instalação: 
Para instalar o software, basta se matar
```sh
$ git clone https://github.com/northy/PergaBot.git
```
Baixar as dependências de bibliotecas
```sh
$ pip install -r requirements.txt
```
E baixar o driver desejado (links abaixo)
